# Douyin Research

适用于公开视频、搜索结果、账号页、热点观察。

Workflow:
- 先确认目标账号、视频、关键词或趋势主题。
- 优先从公开视频、搜索结果、账号页提取证据，不要假设私域数据可见。
- 明确区分公开视频可见信息与需要登录后才能确认的信息。
